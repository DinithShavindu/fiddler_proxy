🛟 Common problems

Quick fixes below.

🌐Websites won't load: double-check the proxy IP and port match exactly what you were given, and that Wi-Fi is the right network.
🔒HTTPS pages show errors (iOS): you missed the trust toggle. Settings → General → About → Certificate Trust Settings → turn it ON.
📌One app shows nothing but the browser works: that app blocks proxies ("certificate pinning"). Nothing you can change — let your team know.
↩️Want to go back to normal: set the Wi-Fi proxy back to None / Off. Everything returns to how it was.

💬 Frequently asked questions

Tap a question to see the answer.

?Is this safe? Can my team see my passwords?▾
While the proxy is on, your team can see the network traffic from this device — that's the whole point of testing. Only use it on a work/test device, and turn the proxy back to None / Off when you're finished. Don't do personal banking while it's active.
?Do I need to install any app?▾
No. You only change Wi-Fi settings and install one small certificate from the browser. Nothing to download from an app store.
?The download page won't open in my browser▾
Make sure you finished Step 1 first (the proxy must be set), you're on the correct Wi-Fi, and you typed the address exactly, e.g. http://192.168.8.102:8866 — with http, not https.
?Android says "install from storage" is blocked▾
Set a screen-lock PIN or password first (Settings → Security → Screen lock). Android requires a lock before it will trust a CA certificate. Then install again.
?HTTPS sites show a security error (iPhone)▾
You missed the trust toggle. Go to Settings → General → About → Certificate Trust Settings and switch the Fiddler certificate ON. This is separate from installing it.
?The browser works but one app shows nothing▾
That app uses "certificate pinning" — it refuses any proxy for security. This can't be fixed from the device; let your team know so they can use a special test build of the app.
?How do I completely undo everything?▾
1) Set the Wi-Fi proxy back to None/Off. 2) Delete the Fiddler certificate — Android: Settings → Security → Encryption & Credentials → User credentials; iOS: Settings → General → VPN & Device Management → remove the profile.
?Nothing loads at all after setup▾
The computer running the proxy is probably off or asleep, or you're on a different Wi-Fi. Make sure that computer is on and you're on the same network. If you just need internet now, set the proxy to None/Off.
